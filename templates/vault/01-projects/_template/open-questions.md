# <Project Name> — Open Questions

Numbered, append-only. When a question is resolved, mark in place by appending `— RESOLVED YYYY-MM-DD` to its heading. Don't delete or renumber.

---

## Q1 — Example question (HIGH)

**What:** One sentence stating the question.

**Why it matters:** Who's blocked by this / what breaks if it stays unresolved.

**Options:**
1. Option one — pros / cons / effort
2. Option two — pros / cons / effort
3. Option three — pros / cons / effort

**Recommended:** The one to pick and why.

---

## Format for new questions

Copy this template at the bottom of the file. Number sequentially.

```markdown
## QN — Short title (SEVERITY)

**What:**

**Why it matters:**

**Options:**
1.
2.

**Recommended:**
```

Severity levels: **CRITICAL** (blocks production), **HIGH** (blocks current milestone), **MEDIUM** (nice to fix), **LOW** (defer), **deferred** (accepted as-is for now).
